## Duplicate scripts Description
This bash script utilize RMAN duplicate command to duplicate, or clone, a database from the backup taken by using backup-ora-coh-sbt.bash script if it is Oracle database on Linix or aix-backup-ora-coh-sbt.bash script if it is Oracle database on AIX.
